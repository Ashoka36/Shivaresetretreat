# 🕉️ Shiva's Reset Retreat - Aurora Borealis Website

A stunning, fully responsive website for **Shiva's Reset Retreat** with an **Aurora Borealis** color theme. This website features smooth animations, mobile-friendly design, and easy-to-customize photo placeholders.

## 🌌 Features

### Design & Aesthetics
- **Aurora Borealis Theme** - Cyan, green, purple, and pink color palette
- **Responsive Design** - Fully optimized for mobile, tablet, and desktop
- **Glass Morphism Effects** - Modern frosted glass navbar
- **Smooth Animations** - Fade-in, float, and glow effects
- **Mobile Menu** - Hamburger navigation for small screens

### Sections
1. **Hero Section** - Eye-catching introduction with animated title
2. **Mind Reset** - Yoga and meditation experience
3. **Restopub & Boba Bar** - Coastal beverage experience
4. **Ground Yourself** - Bhajan concert and wellness
5. **Pet Parent Retreat** - Animal welfare packages
6. **Gallery** - Photo showcase section
7. **CTA Section** - Secure booking button
8. **Footer** - Contact information

### Photo Placeholders
All sections include easy-to-replace photo placeholders:
- Yoga Session image
- Restopub/Boba Bar image
- Bhajan Concert image
- Pet Parent image
- Gallery Collage image

Simply replace the placeholder URLs with your own images!

## 🚀 Quick Start

### Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/Ashoka36/aurora-retreat.git
   cd aurora-retreat
   ```

2. Start a local server:
   ```bash
   python3 -m http.server 8000
   ```

3. Open `http://localhost:8000` in your browser

### Customization

#### Replace Logo
Replace `logo.jpg` with your own logo file (recommended size: 200x200px)

#### Replace Photos
Update the image URLs in `index.html`:
```html
<!-- Change this -->
<img src="https://via.placeholder.com/600x400/1a4d5c/7df9ff?text=Yoga+Session" 
     alt="Yoga Session" class="section-image">

<!-- To this -->
<img src="path/to/your/yoga-image.jpg" 
     alt="Yoga Session" class="section-image">
```

#### Customize Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --aurora-green: #00ff88;
    --aurora-cyan: #7df9ff;
    --aurora-purple: #ff00ff;
    --aurora-pink: #ff7fdb;
    --aurora-blue: #1a4d5c;
    --dark-bg: #0a0e27;
}
```

#### Update Contact Info
Contact information is securely masked in the website. Users click "Book Your Retreat" to reveal contact details.
